## 📚 Description
This project contains three games created using Python and Pygame:
- Racer
- Snake
- Paint

The project is based on lecture examples and extended tutorials.

---

## 🚗 1. Racer

Extended from CodersLegacy tutorial.

### Added features:
- Randomly appearing coins on the road
- Coin collection system
- Display of collected coins in the top right corner
- Increasing game speed over time

---

## 🐍 2. Snake

Extended from lecture example.

### Added features:
- Border collision detection
- Wall collision detection
- Food does not spawn on snake or walls
- Levels system
- Speed increases with level
- Score and level counters

---

## 🎨 3. Paint

Extended from Nerd Paradise tutorial.

### Added features:
- Drawing rectangles
- Drawing circles
- Eraser tool
- Color selection
